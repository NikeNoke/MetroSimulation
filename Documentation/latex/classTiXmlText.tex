\doxysubsection{Detailed Description}
XML text. A text node can have 2 ways to output the next. \char`\"{}normal\char`\"{} output and CDATA. It will default to the mode it was parsed from the XML file and you generally want to leave it alone, but you can change the output mode with \mbox{\hyperlink{classTiXmlText_acb17ff7c5d09b2c839393445a3de5ea9}{Set\+CDATA()}} and query it with \mbox{\hyperlink{classTiXmlText_aac1f4764d220ed6bf809b16dfcb6b45a}{CDATA()}}.
