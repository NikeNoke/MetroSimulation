\doxysubsection{Detailed Description}
Print to memory functionality. The \mbox{\hyperlink{classTiXmlPrinter}{Ti\+Xml\+Printer}} is useful when you need to\+:


\begin{DoxyEnumerate}
\item Print to memory (especially in non-\/\+STL mode)
\item Control formatting (line endings, etc.)
\end{DoxyEnumerate}

When constructed, the \mbox{\hyperlink{classTiXmlPrinter}{Ti\+Xml\+Printer}} is in its default \char`\"{}pretty printing\char`\"{} mode. Before calling Accept() you can call methods to control the printing of the XML document. After \mbox{\hyperlink{classTiXmlNode_acc0f88b7462c6cb73809d410a4f5bb86}{Ti\+Xml\+Node\+::\+Accept()}} is called, the printed document can be accessed via the \mbox{\hyperlink{classTiXmlPrinter_a859eede9597d3e0355b77757be48735e}{CStr()}}, Str(), and \mbox{\hyperlink{classTiXmlPrinter_ad01375ae9199bd2f48252eaddce3039d}{Size()}} methods.

\mbox{\hyperlink{classTiXmlPrinter}{Ti\+Xml\+Printer}} uses the Visitor API. \begin{DoxyVerb}TiXmlPrinter printer;
printer.SetIndent( "\t" );

doc.Accept( &printer );
fprintf( stdout, "%s", printer.CStr() );
\end{DoxyVerb}
