\doxysubsection{Detailed Description}
Klasse \mbox{\hyperlink{classStation}{Station}}. Een station is een object waarover \mbox{\hyperlink{classTram}{Tram}} objecten zouden rijden. \mbox{\hyperlink{classStation}{Station}} objecten vormen een spoornummer, beter gezegd, een lijn in het \mbox{\hyperlink{classMetroNet}{Metro\+Net}}. Stations zijn met elkaar verbonden, elk station heeft een volgende station en vorige station.
