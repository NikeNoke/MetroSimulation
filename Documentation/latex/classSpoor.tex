\doxysubsection{Detailed Description}
Klasse \doxylink{classSpoor}{Spoor}. Een spoor maakt deel uit van een station. Een station kan meerdere sporen hebben waarmee we meerdere lijnen kunnen hebben die door één station komen. Dit zijn de sporen.
