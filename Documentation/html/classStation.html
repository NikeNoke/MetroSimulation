Detailed Description

Klasse Station. Een station is een object waarover Tram objecten zouden rijden. Station objecten vormen een spoornummer, beter gezegd, een lijn in het MetroNet. Stations zijn met elkaar verbonden, elk station heeft een fVolgende station en fVorige station.
